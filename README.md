# Coin Toss Game (CLI)

A simple menu-based Python terminal game where you guess **Heads** or **Tails** and track your **wins/losses** over time.

## Features
- Toss a virtual coin (Heads/Tails)
- Guess the outcome
- Scoreboard: wins and losses
- Simple CLI menu navigation

## Requirements
- Python 3.x

## Run
```bash
python main.py
